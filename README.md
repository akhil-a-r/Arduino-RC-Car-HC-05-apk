This project explores building a Bluetooth controlled car using an Arduino Uno microcontroller, an HC-05 Bluetooth module, and a buzzer .

Components:

Arduino Uno: A Microcontroller
HC-05 Bluetooth Module: Bridges the gap between your phone and the Arduino for wireless communication.
DC Motors (2): Power the wheels and propel the car.
Motor Driver (L298N or Similar): Safely manages the high current needed by the motors based on Arduino signals.
Wheel Caster Set (2): Provides steering and maneuverability.
Chassis: The base that holds all the components together (pre-built or DIY).
Power Source: Batteries (e.g., Li-ion) to provide juice for the car.
Jumper Wires: Connect all the electronic components.
Breadboard (Optional): Useful for prototyping the circuit before final assembly.
Buzzer: Adds an auditory element for feedback (e.g., honking sound).
Smartphone with Bluetooth App: Pre-built app or custom-made one to send directional controls (forward, backward, left, right) to the Arduino.
